== Description ==
简单方便的在线小商店!支持在线支付(此插件永久免费使用)
任何使用问题可联系QQ:781272314 反馈(有任何建议也可提)


= 页面地址: =
* 你的域名地址/products (即可查看!如果打开显示404页面,请先到后台 '设置'->'固定链接' 设置成'文章名' )
* 把地址添加到 菜单 自定义链接,就能在前台显示商店了

查看[演示](https://wpapi.aliluv.cn/products/)




== Installation ==

1. 上传 `jiutu-qingdong`目录 到 `/wp-content/plugins/` 目录或后台搜索`情动`
2. 在后台插件菜单激活该插件
